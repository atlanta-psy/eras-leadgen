# агент лидогенерации Эрас — как работает

программа на компьютере. собирает контакты бизнесов в нише, чистит, пишет персональные офферы и помогает безопасно рассылать с учётом лимитов и фоллоу-апов. отправляете вы руками — агент не логинится в ваши аккаунты.

## установка (один раз)
```
python3 -m pip install pyyaml
```
папку `leadgen` положить куда удобно, команды запускать из неё.

## конвейер — 4 шага

```
# 1. собрать лидов из openstreetmap по нише
python3 leadgen.py collect --niche guest_house

# 2. обогатить: зайти на сайты, найти email/телеграм/вк, понять есть ли бронь, отсеять дубли
python3 enrich.py --niche guest_house

# 3. собрать персональные офферы (тема + сообщение + 2 фоллоу-апа + КП)
python3 offer.py --niche guest_house

# 4. сформировать пачку на сегодня (с прогревом и лимитами)
python3 send_manager.py today --niche guest_house
```

## ежедневная работа

1. утром: `send_manager.py today --niche guest_house` — получаешь файл `today_*.csv` с пачкой на день.
2. отправляешь сообщения из пачки руками (email / телеграм / вк).
3. кто ответил — отмечаешь: `send_manager.py replied --niche guest_house --name "Оливия"`.
4. фоллоу-апы агент подкинет сам в следующие дни тем, кто молчит.
5. смотреть воронку: `send_manager.py stats --niche guest_house`.

## что где править (это контент, не код)

- `configs/guest_house.yaml` — районы поиска и теги ниши.
- `configs/guest_house_offer.yaml` — весь текст оффера: боли, услуги, кейс, промо, контакты, тема письма, фоллоу-апы.
- `configs/send_config.yaml` — дневные лимиты и прогрев аккаунтов.

## новая ниша
скопировать два конфига `guest_house*.yaml` → переименовать (например `restaurant.yaml` и `restaurant_offer.yaml`), вписать теги/районы и текст оффера, и гонять с `--niche restaurant`. движок тот же.

## безопасность
- агент НЕ отправляет сам и не входит в аккаунты — только готовит пачки. отправляете вы.
- холодную рассылку в телеграм/вк — с отдельных, не рабочих аккаунтов, с прогревом (лимиты в `send_config.yaml`).
- почта — объёмный канал; для неё нужен отдельный домен с настройкой (spf/dkim) и строка отписки, иначе письма уходят в спам.
- телефон в рассылку не идёт — только звонок при желании.

## что собрано
сбор + обогащение (email/тг/вк со сайтов, детект готовой брони) + дедуп + персональные офферы с темой и фоллоу-апами + диспетчер с прогревом и трекером.

## что можно добавить дальше
- автоотправку почты через ваш домен (по желанию);
- источник 2гис для объёма;
- красивый pdf-КП через бренд-скил Педро для тёплых лидов;
- парсер вк-сообществ как отдельный источник.
