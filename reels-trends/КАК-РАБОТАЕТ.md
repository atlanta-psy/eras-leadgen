# Reels-разведка по нише (Instagram через Apify)

Открытие трендов: берёт reels по хэштегам твоей ниши, считает вовлечённость (ER) и выдаёт выборку залетевшего — для адаптации под себя. Работает с русской нишей (в отличие от tiktok).

## Шаг 0. Ключ Apify (один раз)
1. зайти на apify.com, зарегистрироваться.
2. Settings → Integrations → скопировать **API token**.
3. вставить его в `config.yaml` в строку `apify_token`.
Оплата — по факту объёма (instagram ~$1,5 за 1000 постов). Маленькие прогоны стоят копейки; на старте обычно хватает бесплатного кредита.

## Шаг 1. Настроить
В `config.yaml`: список `hashtags` (хэштеги ниши), `results_limit` (постов на хэштег), `min_likes`, `max_age_days`, `top`.

## Шаг 2. Запустить
```
pip install pyyaml
python3 ig_reels.py
```
На выходе — два файла с датой:
- `reels-разведка-ДАТА.md` — отчёт: топ reels с метриками и вовлечённостью.
- `reels-разведка-ДАТА.csv` — таблица.

## Что в выборке
По каждому reels: подпись (хук), автор, просмотры, лайки, комменты, **ER%**, хэштег, дата, ссылка.
- высокие просмотры = тема/хук залетели;
- высокий ER = реально цепляет.

Бери хук и первый кадр, адаптируй под свой голос (можно через скил reels-hooks → потом elena-content-engine).

## Заметки
- Это открытие трендов по хэштегам (то, что ты хотела), а не слежка за аккаунтами.
- Расход Apify зависит от `results_limit` × число хэштегов. Начни с малого, потом расширяй.
- Связка по платформам: reels-разведка (инста, ру) + youtube-агент (ру, бесплатно) + tiktok-агент (англо). Тренды отсюда → контент через elena-content-engine.
