# TikTok-разведка по нише

Бесплатный агент: следит за выбранными аккаунтами конкурентов/коллег в TikTok, берёт их свежие ролики, считает просмотры и вовлечённость и выдаёт ранжированную выборку залетевшего — чтобы ты адаптировала под себя.

Работает на том же `yt-dlp`, что и YouTube-агент (он у тебя уже установлен).

## Шаг 1. Вписать аккаунты
Открой `accounts.yaml`, в список `accounts` впиши реальные @-аккаунты из своей ниши (психологи, коучи), убери строки с `example`. Там же настройки: сколько роликов смотреть, окно по свежести, порог просмотров.

## Шаг 2. Запустить
```
python3 tiktok_trends.py
```
Идёт пару минут. На выходе — два файла с сегодняшней датой:
- `tiktok-разведка-ДАТА.md` — отчёт: топ залетевших роликов с метриками и подсказкой, как читать.
- `tiktok-разведка-ДАТА.csv` — та же выборка таблицей.

## Что в выборке
По каждому ролику: подпись (хук), аккаунт, просмотры, лайки, комменты, **вовлечённость (ER%)**, дата, ссылка.
- высокие просмотры = тема/хук залетели;
- высокий ER = ролик реально цепляет, а не просто попал в показы.

Бери заголовок и первый кадр, адаптируй под свой голос.

## Важно / ограничения
- Это слежка за **аккаунтами**, которые ты задала. Открытого поиска «что вообще трендит по хэштегу» в бесплатном TikTok нет — это уже платный скрейпер (можем добавить позже).
- Чем больше аккаунтов впишешь, тем шире картина. 15-30 — хороший охват ниши.
- Если yt-dlp начнёт ругаться — обнови его: `python3 -m pip install -U yt-dlp`.
